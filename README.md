## 中文介绍 / Chinese introduction

本仓库是 [IcePatch](https://github.com/Anatdx/IcePatch) 的文档库，网页版本请参阅[此处](https://icepatch.anatdx.com)。

**IcePatch 是 APatch 的 fork 项目。**在保持原有功能与兼容性的基础上进行持续维护与扩展。

目前本仓库的文档有[简体中文](/docs/zh_CN)、[英语](/docs/)、[葡萄牙语（巴西）](/docs/pt_BR)、[俄语](/docs/ru)等语言的版本，欢迎翻译并贡献本仓库文档的其他语言版本。我们需要你的帮助。

## English introduction

This repository is for [IcePatch Documentation](https://icepatch.anatdx.com), which is created for [IcePatch Project](https://github.com/Anatdx/IcePatch).

**IcePatch is a fork of APatch.** It is maintained and evolved while preserving behavior and compatibility where possible.

**Currently there are only [zh_CN](/docs/zh_CN), [EN](/docs/), [pt_BR](/docs/pt_BR), [RU](/docs/ru) documents, and you are welcome to contribute versions in other languages. We need you.**
